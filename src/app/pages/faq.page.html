Frequently Asked Questions (FAQs)

Why is the analysis so slow?

The program runs on a shared virtual server, with limited memory and CPU. Upgrading the server would improve performance, but would be rather expensive for me.

How can I contribute?

The application is open source and available on GitHub. If you have any suggestions or improvements, please feel free to open an issue or pull request.

What's the technical background of this application?

The application is built using Angular and TypeScript, with a backend API written in Java with Spring Boot, using JNA to wrap the native C++ library. The frontend is served by a static file server, and the backend API is hosted on a shared virtual server.

What are the decimal numbers after the player labels?

The decimal numbers show the hand evaluation according to the Kaplan-Rubens hand evaluation algorithm. See Richard Pavlicek.

What are planned features for this site?

As a next feature I plan to implement a hand generation tool, suitable for generating hands for partnership bidding practice.
